# LGORITHMIC-TRADING-MID-PRICE-VOLATILITY-PREDICTION
Predict future mid-price and price direction using high-frequency orderbook data. Incorporates volatility modeling, AI/ML techniques, statistical formulas, and visual evaluation to justify trade decisions and validate prediction accuracy.


##Key Features

Mid-Price Prediction: Forecast the next mid-price from orderbook data.
Price Direction Classification: Predict whether the price will move up or down.
Volatility Modeling: Use EWMA and other measures to guide trade decisions.
Statistical & Mathematical Explanations: Document formulas and methods used.
Evaluation & Visualization: Quantile analysis, correlation checks, and plots to validate predictions.

##Dataset

Original data in .txt format, converted to .csv.
